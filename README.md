# Good Question website

A complete static website ready for Vercel, Netlify, GitHub Pages or any ordinary web hosting.

## Vercel
1. Upload this folder to a GitHub repository.
2. Import the repository into Vercel.
3. Framework preset: **Other**.
4. Leave Build Command and Output Directory empty.
5. Deploy.

The main page is fully self-contained: layout, styles, animations and both logo versions are embedded directly in `index.html`.
